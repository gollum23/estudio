Introducción django

Django

El framework web para perfeccionistas con fechas límite

Django hace que sea más fácil construir mejores aplicaciones web más rápido y con menos código

https://www.djangoproject.com/

Caracteristicas

ORM: Defina sus modelos de datos completamente en python, usted tiene una completa API dinamica para acceso a la base de datos.

Interface de Administración automatica: Ahorrese el trabajo aburrido de crear interfaces para que los clientes agregen y actualicen contenido

Sistema de plantillas: Use el poderoso, escalable y amigable lenguaje de plantillas para separar el diseño, contenido y codigo Python

Sistema de cache: Trabajando con memcached u otro framwork de cache, el cache puede ser tan granular como usted quiera

Internacionalización: Tiene soporte para aplicaciones multilenguaje.

Instalación

Instalar django es tan sencillo como abrir la terminal y escribir

                            pip install django

para verificar si django esta instalado correctamente en la terminal escribimos

                            python
                            import django
                            django.VERSION
                            # (1, 5, 1, 'final', 0)

Crear un proyecto

Para crear un proyecto de django es tan sencillo como desde la consola navegar hasta el directorio donde vamos a trabajar nuestro proyecto, posteriormente escribir en la consola

                        django-admin.py startproject miproyecto

esto nos crea una estructura de directorios así

                        miproyecto/
                            manage.py
                            miproyecto/
                                __init__.py
                                settings.py
                                urls.py
                                wsgi.py

El archivo settings.py contiene la configuración de nuestro proyecto como datos de conexión a la base de datos, carpetas static y media, aplicaciones instaladas en el proyecto

El archivo urls.py es nuestro enrutador el es el que verifica la url ingresada en el navegador y la redirije a la vista correcta.

Crear una aplicación

Dentro de las filosofias django esta el no escribir código de más, por esto django divide las partes de un proyecto en aplicaciones, con esto se modularizan los proyecto y se reutiliza el codigo de la aplicación en otro proyecto.

Para crear una nueva aplicación digitamos en consola

                        django-admin.py startapp nombreapp

Nuestra estructura de archivos ahora se vera así:

                        miproyecto/
                            manage.py
                            miproyecto/
                                __init__.py
                                settings.py
                                urls.py
                                wsgi.py
                            nombreapp/
                                __init__.py
                                models.py
                                tests.py
                                views.py
