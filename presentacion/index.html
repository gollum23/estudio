¿Qué es Python?

Python es un lenguaje de programación similar a Perl, pero con una sintaxis muy limpia y ayuda a un código legible.

Es un lenguaje interpretado o de script, con tipado dinámico, fuertemente tipado, multiplataforma y orientado a objetos

Lenguaje interpretado o de script

Un lenguaje interpretado o de script es aquel que se ejecuta utilizando un programa intermedio llamado intérprete, en lugar de compilar el código a lenguaje máquina que pueda ejecutar directamente una computadora

Los lenguajes compilados son más rapidos en su ejecución, pero los lenguajes interpretados son más flexibles y portables

Python sin embargo cuenta con las características de los lenguajes compilados ya que el codigo fuente se traduce a un pseudo código máquina llamado bytecode en su primera ejecución, generando archivos con extención .pyc o .pyo (bytecode optimizado)

Tipado dinámico

El tipado dinámico se refiere a que no es necesario declarar el tipo de dato que va a ser almacenada en una variable, y puede cambiar de tipo de dato según el tipo del valor que se asigne


Fuertemente tipado

No se permite tratar a una variable como si fuera de un tipo distinto al que tiene

Si tenemos una variable de tipo string no podemos concatenarla con una variable de tipo numero

Multiplataforma

El intérprete de Python está disponible para la mayoria de plataformas (UNIX, Solaris, Linux, DOS, Windows, OS/2, Mac Os) por lo que si no se utilizan librerias especificas de cada plataforma el programá podrá correr en todos los sistemas sin hacer mayores modificaciones

Orientado a Objetos

La orientación a objetos es un paradigma de programación en el que los conceptos del mundo real relevantes para solucionar nuestro problema se trasladan a clases y objetos

Python también permite la programación imperativa, funcional y orientada a aspectos

¿Por qué Python?

Su sintaxis simple, clara y sencilla.

Tipado dinámico, gestor de memoria.

Gran cantidad de librerías disponibles y la potencia del lenguaje, hacen que desarrollar en Python sea sencillo, rápido y sobretodo divertido.

La sintaxis es tan sencilla y parecida al lenguaje natural que los programas elaborados en Python parecen pseudocódigo. Por este motivo es uno de los mejores lenguajes para aprender a programar.

Tipos de datos básicos

En Python encontramos 3 tipos de datos básicos

  • Números (Enteros, Coma Flotante y Complejos)
  • Cadenas de texto (Strings)
  • Booleanos (True y False)

Operadores Numéricos

Operador Descripción Ejemplo
+ Suma r = 3 + 2 # r es 5
- Resta r = 4 - 7 # r es -3
- Negación r = -7 # r es -7
* Multiplicación r = 2 * 6 # r es 12
** Exponente r = 2 ** 6 # r es 64
/ División r = 3.5 / 2 # r es 1.75
// División entera r = 3.5 // 2 # r es 1.0
% Modulo r = 7 % 2 # r es 1

Consideraciones

Es importante tener en cuenta que si usamos dos números enteros en la división (/) python entendera que queremos que el resultado sea un entero por consiguiente 3 / 2 = 1, si queremos que de un resultado decimal tenemos que hacer 3.0 / 2 o hacer un casteo de datos asi

   r = float(3) / 2

Esto es así porque cuando se mezclan tipos de números, Python convierte todos los operandos al tipo más complejo entre los tipos de los operandos

Cadenas

Las cadenas de texto no es nada diferente que texto encerrado entre comillas simples o dobles, dentro de las comillas se pueden utilizar caracteres especiales escapandolos con \, como \n carácter de nueva línea

Una cadena puede estar precedida por el carácter u o r, los cuales indican que se trata de una cadena con codificación Unicode o una cadena raw(cruda) respectivamente. Las cadenas raw se distingen de las normales en que los caracteres especiales escapados con \ no se sustituye por sus contrapartidas, esto es útil para expreciones regulares (Lo veremos en django)

Las cadenas entre comillas triples """cadena""" o '''cadena''' permiten escribir el texto en multiples lineas

Cadenas

El operador + funciona para concatenar las cadenas.

                         a = "uno"
                         b = "dos"
                         c = a + b
                         #c es "unodos"

El operador * repite la cadena el numero de veces que indique el número utilizado como segundo operando.

                         a = "uno"
                         c = a * 3
                         #c es "unounouno"

Booleanos

Los valores que puede tener una variable de tipo booleana sólo son True (cierto) o False (falso), son utilizadas para expresiones condicionales y bucles

Operador Descripción Ejemplo
and ¿se cumple a y b? r = True and False # r es False
or ¿se cumble a o b? r = True or False # r es True
not No a r = not True # r es False

Booleanos

Los valores booleanos son además el resultado de expresiones que utilizan operadores relaciones

Operador Descripción Ejemplo
== ¿Son iguales a y b? r = 5==3 # r es False
!= ¿Son distintos a y b? r = 5 != 3 # r es True
< ¿es a menor que b? r = 5 < 3 # r es False
> ¿es a mayor que b? r = 5 > 3 # r es True
<= ¿es a menor o igual que b? r = 5 <= 5 # r es True
>= ¿es a mayor o igual que b? r = 5 >=3 # r es True

Colecciones

Listas

La lista es un tipo de colección ordenada, lo que en otros lenguajes se conoce por arrays o vectores

Las listas pueden contener cualquier tipo de dato: números, cadenas, booleanos e inclusive listas

Crear una lista es tan sencillo como indicar entre corchetes, y separados por comas, los valores que queremos incluir

                        l = [22, True, "una lista", [1, 2]]

Para acceder a un valor de la lista escribimos el nombre de la lista y el índice del elemento entre corchetes teniendo en cuenta que los índices inician en 0

                        mi_var = l[1] # mi_var = True

Listas

Para acceder a un elemento de una lista incluida dentro de otra lista solo se tendra que usar dos veces el operador, el primero indica la posición de la lista exterior y el segundo para seleccionar el elemento de la lista interior

                        l = [22, True, "una lista", [1, 2]]
                        mivar = l[3][1] # mivar = 2

Tambien podemos usar este operador para modificar un elemento de la lista si se coloca en la parte izquierda de una asignación

                        l = [22, True, "una lista", [1, 2]]
                        l[0] = 99 # l sera [99, True, "unalista", [1, 2]]

listas

Una curiosidad del operador [] es que se puede utilizar con un valor negativo, lo que hara es comenzar el índice desde el final, es decir [-1] y asi sucesivamente

Otro funcionalidad de este operador es conocida como slicing, consiste en seleccionar porciones de la lista, si en lugar de escribir un numero escribimos dos separados por : (inicio:fin) indicamos que queremos una lista que vaya desde la posición de inicio hasta la posición de fin sin incluir este último. Si escribimos tres numeros (inicio:fin:salto), el tercer numero se utiliza para determinar cada cuantas posiciones añadir un elemento a la lista

                        l = [22, True, "una lista", [1, 2]]
                        mivar = l[0:2] # mivar es igual a [22, True]
                        mivar = l[0:4:2] # mivar es igual a [22, "una lista"]
Se pueden usar tambien numeros negativos con el comportamiento mencionado anteriormente

Tuplas

Todo lo visto para las listas aplica para las tuplas a excepción de la forma de definirla, ya que se utilizan paréntesis en lugar de corchetes

                        l = (22, True, "una lista",)

Es necesario tener en cuenta que para las tuplas de un solo elemento es necesario añadir una coma para diferenciarlo de un elemento entre parentesis

                        l = (22, )
                        type(l) # type tupla
                        l = (22)
                        type(l) # type int

La diferencia entre las tuplas y las listas es que las primeras no tienen mecanismos de modificación, son inmutables, es decir sus valores no se pueden modificar despues de ser creada

Tuplas

A cambio de estas limitaciones las tuplas son mas ligeras

Como una buena practica se recomienta poner siempre la coma al final del último elemento de una tupla

                        l = (22, True, "una lista",)

Diccionarios

Los diccionarios también llamados matrices asociativas, deben su nombre a que son colecciones que relacionan una clave y un valor similar a la notación JSON

                        d = {"Love Actually": "Richard Curtis",
                                 "Kill Bill": "Tarantino",
                                 "Amélie": "Jean-Piere >Jeunet"}

El primer valor es la clave y el segundo el valor asociado a este

La principal diferencia entre los diccionarios y las listas o tuplas es que no se accede a los valores por su índice, sino por su clave utilizando nuevamente el operador []

                        d["Love Actually"] #devuelve "Richard Curtis"

Diccionarios

Al igual que en las listas se puede utilizar este operador para reasignar valores

                        d["Kill Bill"] = "Quentin Tarantino"

En los diccionarios no se puede utilizar slicing debido a que los diccionarios no son secuencias si no mappings

Control de flujo

Sentencias condicionales

Si un programa no fuera más que una lista de instrucciones que se ejecutan de forma secuencial no tendría mucha utilidad. Los condicionales nos permiten comprobar condiciones y hacer que nuestro programa se comporte de una y otra forma dependiendo de esta condición.

Aquí es donde se hacen importante el tipo de dato Booleano y los operadores relacionales.

if

Es la forma más simple de un condicional, su sintaxis es un if seguido de la condición a evaluar, dos puntos (:) y en la siguiente linea e indentando el codigo que se ejecuta si se cumple esa condicion

                        fav = "github.com"
                        if fav == "github.com":
                            print "Usas el mejor sistema de versiones"

if..else

Este es un condicional un poco más complicado, basicamente evalua si se se cumple la condición ejecuta un código y si no ejecuta otro.

            fav = "github.com"
            if fav == "github.com":
                print "Usas el mejor sistema de versiones"
            else:
                print "No usas ningun sistema de versiones, que lastima"

if... elif... else

En python no existe la construccion switch para eso se usa la construccion elif que es la contracción de else if, lo que hace es evaluar la primera condicion si no se cumple pasa al else if que es otra condicion y si tampoco se cumple y pasa al siguiente, si ninguna se cumple ejecutara el codigo del else

                        if numero < 0:
                            print "Negativo"
                        elif numero > 0:
                            print "Positivo"
                        else:
                            print "Cero"

A if C else B

Existe adicionalmente una construcción similar al operador ? de otros lenguajes, no es mas que una forma compacta de un if else.

En esta esta construcción se evalua C, si se cumple devuelve A, en caso contrario devuelve B

                        var = "par" if (num % 2 == 0) else "impar"

Control de flujo

Bucles

Los condicionales nos permiten ejecutar diferentes fragmentos de código dependiendo de ciertas condiciones, los bucles permiten ejecutar cierto número de veces un fragmento de código mientras se cumpla una condición

while

El bucle while(mientras) ejecuta un fragmento de código mientras se cumpla una condición

                        edad = 0
                        while edad < 18:
                            edad = edad + 1
                            print "Felicidades, tienes " + str(edad)

for in

En python for se utiliza como una forma genérica de iterar sobre una secuencia, el siguiente ejemplo muestra el aspecto del bucle for en Python

                        secuencia = ["uno", "dos", "tres"]
                        for elemento in secuencia:
                            print elemento

Leyendo en lenguaje natural la cabecera del bucle seria, "Para cada elemento en secuencia"

¿qué ocurre si quisiéramos utilizar el for para imprimir los numeros del 30 a 50?

Python tiene una funcion llamada range

                        for i in range (30, 51):
                            print i

Esto nos mostrara una lista con los números desde el 30 hasta el 50

Funciones

Una función es un fragmento de código con un nombre asociado que realiza una serie de tareas y retorna un valor. A los fragmentos de código que no devuelven valores se les llama procedimientos.

En Pyhon no existen los procedimientos, ya que cuando no se especifica un valor de retorno la función devuelve el valor None (nada)

Además de ayudarnos a programar y depurar dividiendo el programa en partes, las funciones también permiten reutilizar código

Definición de una función

En Python la funcion se declara con la palabra clave def seguida del nombre de la función y entre paréntesis los argumentos separados por comas. A continuación, en otra línea, indentado y despues de los dos puntos se tienen las líneas de código que conforman la función

                        def mi_funcion(par1, par2):
                            print par1
                            print par2

Tambien se puede encontrar una cadena de texto como primera linea del cuerpo de la funcion, estas cadenas se conocen como docstring y funcionan a modo de documentación de la función

        def mi_funcion(par1, par2):
            """Esta funcion imprime los valores pasados como parametros"""
            print par1
            print par2

Orientación a objetos

Clases y Objetos

Un objeto es una entidad que agrupa un estado y una funcionalidad relacionadas. El estado del objeto se define por medio de variables llamadas atributos, mientras que la funcionalidad se modela a través de funciones que se conocen como métodos

Una clase, por otro lado, no es mas que una plantilla genérica a partir de la cuál instanciar los objetos.

Clases

En Python las clases se definen mediante la palabra clave class seguida del nombre de la clase y dos puntos (:), a continuación indentando el cuerpo de la clase

            class Coche:
                """Abstracción de los objetos coche."""
                def __init__(self, gasolina):
                    self.gasolina = gasolina
                    print "Tenemos", gasolina,  "litros"

                def arrancar(self):
                    if self.gasolina > 0:
                        print "Arranca"
                    else:
                        print "No arranca"

                def conducir(self):
                    if self.gasolina > 0:
                        self.gasolina -= 1
                        print "Quedan", self.gasolina,  "litros"
                    else:
                        print "No se mueve"

Lo primero que podemos ver es el método __init__, este nombre es una convención y se ejecuta justo después de crear un nuevo objeto a partir de la clase, proceso que se conoce como instanciación. El método __init__ como su nombre lo indica, sirve para realizar cualquier proceso de inicialización que se nacesario

El primer parametro de __init__ y del resto de métodos es self y sirve para referirse al objeto actual.

Con el ejemplo anterior para crear un objeto se escribría el nombre de la clase seguido de cualquier parámetro que sea necesario entre paréntesis.

                        mi coche = Coche(3)

Herencia

En un lenguaje orientado a objetos cuando una clase hereda de otra clase estamos haciendo que la subclase contenga todos los atributos y métodos del la super clase


Herencia multiple

En python, a diferencia de otros lenguajes se permite la herencia multiple

Polimorfismo

Python, al ser de tipado dinámico, no impone restricciones a los tipos que se le pueden pasar a una función. Por este motivo, a diferencia de Java o C++, el polimorfismo en Python no es de gran importancia


Encapsulación

En Python no existen modificadores de acceso, lo que se hace es que el acceso a una variable o función viene determinado por su nombre, si comienza con dos giones bajos y no termina con dos guiones bajos se trata de una variable o función privada.

Los métodos que inician y terminan con dos guiones bajos son métodos especiales de Python que llama automáticamente bajo ciertas circunstancias.

Decoradores

Un decorador no es mas que una función que recibe una función como parámetro y devuelve otra función como resultado


Funciones lambda

El operador lambda sirve para crear funciones anónimas en línea, por ser anónimas estas no podrán ser referenciadas más tarde

                    l = [1, 2, 3]
                    l2 = filter(lambda n: n % 2.0 == 0, l)
La funcion filter verifica que los elementos de una secuencia cumpla una determinada condición, devolviendo una secuencia con los elementos que cumplen esta condición.

Guia de estilo (PEP8)

En Python existe una guia de estilo que permite la estandarización en la escritura de código las más relevantes son

  • Usar espacios y no tabulaciones para la indentación
  • Usar 4 espacios para cada nivel de indentación
  • Nombres de funciones y variables inician con minusculas
  • Nombres de Clases inician con Mayusculas
  • Linea máxima de 79 caracteres

Crear un proyecto

Para crear un proyecto de django es tan sencillo como desde la consola navegar hasta el directorio donde vamos a trabajar nuestro proyecto, posteriormente escribir en la consola

                        django-admin.py startproject miproyecto

esto nos crea una estructura de directorios así

                        miproyecto/
                            manage.py
                            miproyecto/
                                __init__.py
                                settings.py
                                urls.py
                                wsgi.py

El archivo settings.py contiene la configuración de nuestro proyecto como datos de conexión a la base de datos, carpetas static y media, aplicaciones instaladas en el proyecto

El archivo urls.py es nuestro enrutador el es el que verifica la url ingresada en el navegador y la redirije a la vista correcta.
